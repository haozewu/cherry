# 文件导航顺序说明

### 文件目录说明：

在markdown文件头部 yaml设置目录名。 如果没有一级导航，则无需设置。 目录的排序是在 `cherry.config.js` 中设置。如下：

```js
category: [
        "目录一", "目录二", "目录三"
    ],
```

### 目录中每个显示文件的排序

依据此md文件在目录中的顺序。 如果想配置目录名建议使用如下规则：

```
目录名
    -- a_file.md
    -- b_file.md
    -- c_file.md
```
> `注：目录名非文件夹名`
